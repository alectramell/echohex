# ECHOHEX v1.0

## INSTALL:

		$ sudo dpkg -i <(curl -s https://github.com/alectramell/echohex/raw/master/echohex-1.0.deb)

## USAGE:

	__STRING TO HEX..__

		$ echohex "hello world"

	__HEX TO STRING..__

		$ showhex "68656c6c6f20776f726c64"
